Homework 01 — Многостраничный сайт

Создайте проект из 5 страниц на чистом HTML: главная, о себе, увлечения, галерея, контакты. Используйте семантические теги, единое меню и валидный HTML5.

Общие требования

HTML-структура

  1. Валидный HTML5 (проверьте на validator.w3.org).
  2. Семантические теги везде, где возможно.
  3. Правильная иерархия заголовков (один h1 на страницу и далее h2–h3).
  4. Alt-атрибуты для всех изображений.
  5. Комментарии в коде для основных разделов.

Навигация

  1. Единое меню на всех страницах.
  2. Активная страница выделена (например, class="active" на ссылке).
  3. Хлебные крошки на внутренних страницах (например: Главная → Раздел).

Контент

  1. Минимум 300 слов на каждой странице.
  2. Разнообразные элементы: списки, таблицы, изображения, цитаты, ссылки, формы и т.д.
  3. Реальная или правдоподобная информация.

Частые ошибки

Что НЕ нужно делать

  • Подключать стили внутри HTML (в этом задании — без CSS).
  • Копировать чужой код без понимания.
  • Использовать устаревшие теги: <center>, <font> и т.п.
  • Оставлять Lorem Ipsum вместо реального текста.
  • Забывать закрывающие теги.
  • Использовать пробелы в именах файлов.

Что НУЖНО делать

  • Проверять код в валидаторе.
  • Тестировать все ссылки.
  • Добавлять комментарии.
  • Использовать осмысленные имена файлов.
  • Организовать файлы в папки при необходимости.

Бонусные задания

  1. Микроразметка: Schema.org, Open Graph теги.
  2. Доступность: ARIA-атрибуты, структура для скринридеров, skip navigation link.
  3. SVG-графика: простая SVG-иконка, inline SVG.
  4. Прогрессивное улучшение: сайт работает без CSS/JS/изображений.

Подсказки

  1. Нарисуйте структуру на бумаге.
  2. Составьте список контента.
  3. Соберите материалы (тексты, фото).
  4. Создайте базовую структуру файлов.
  5. Постепенно добавляйте контент.

Шаблон страницы

<!DOCTYPE html>
<html lang="ru">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta name="description" content="Описание страницы">
  <meta name="author" content="Ваше имя">
  <title>Заголовок страницы</title>
</head>
<body>
  <!-- Шапка сайта -->
  <header>
    <nav aria-label="Основное меню">
      <ul>
        <li><a href="index.html" class="active">Главная</a></li>
        <li><a href="about.html">О себе</a></li>
        <li><a href="hobbies.html">Увлечения</a></li>
        <li><a href="gallery.html">Галерея</a></li>
        <li><a href="contacts.html">Контакты</a></li>
      </ul>
    </nav>
    <nav aria-label="Хлебные крошки">
      <ol>
        <li><a href="index.html">Главная</a></li>
        <li aria-current="page">Текущая страница</li>
      </ol>
    </nav>
  </header>

  <!-- Основное содержимое -->
  <main id="content">
    <!-- Ваш контент здесь (минимум 300 слов) -->
  </main>

  <!-- Подвал сайта -->
  <footer>
    <p>&copy; 2025 Ваше имя. Все права защищены.</p>
  </footer>
</body>
</html>

FAQ

Можно ли использовать CSS?

В этом задании — нет. CSS будет в следующем.

Откуда брать картинки?

Свои фото или бесплатные стоки: Unsplash, Pexels. Допустимы заглушки Placeholder.

Что если не хочу писать о себе?

Можно сделать сайт о вымышленном персонаже, но соблюсти требования.

Сколько должно быть текста?

Минимум 300 слов на страницу.

Можно ли использовать JavaScript?

Нет, только HTML.
